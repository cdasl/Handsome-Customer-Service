class backend.models.Customer(*args, **kwargs)¶

客服表

  • CID(String)：客服ID
  • EID(String)：客服所在企业ID
  • name(String)：客服昵称
  • state(int)：客服状态（-1：注销，0：未激活，1：激活，2：离线，3：在线）
  • service_number(int)：客服正在服务的人数
  • serviced_number(int)：客服服务过的人数
  • last_login(date)：客服最后一次登录的时间

class backend.models.Enterprise(*args, **kwargs)¶

企业表

  • EID(String)：企业ID
  • name(String)：企业名字
